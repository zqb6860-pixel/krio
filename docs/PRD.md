## 一、产品概述

LightWords（简词）是一款面向中国英语学习者的单词记忆应用，融合科学记忆算法（SM-2 间隔重复）与沉浸式打字练习，支持 Web 端和移动端（Expo）。

### 目标用户
- 大学生（四六级、考研）
- 雅思/托福备考者
- 高中生（高考英语）
- 英语学习爱好者

## 二、技术架构

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端 API | Node.js + Express + TypeScript + Prisma | RESTful API |
| 数据库 | SQLite（开发）/ PostgreSQL（生产） | Prisma ORM |
| Web 前端 | Next.js 14 + React + Tailwind CSS | App Router |
| 移动端 | Expo 54 + React Native | Expo Router |
| 状态管理 | Zustand | 轻量状态 |
| 认证 | JWT + bcrypt + 手机验证码 + 微信 OAuth | 多登录方式 |
| 记忆算法 | SM-2 间隔重复 | 遗忘曲线 |
| 部署 | Docker Compose | 一键部署 |

## 四、数据模型

### 核心表
- `User` - 用户（邮箱/手机/微信登录）
- `UserSettings` - 用户设置（每日目标、发音偏好等）
- `Word` - 单词（音标、难度、频率）
- `WordMeaning` - 释义（词性、英文定义、中文翻译）
- `WordRoot` - 词根词缀（前缀/后缀/词根、来源、相关词）
- `WordCollocation` - 搭配用法
- `WordExample` - 例句（句子、翻译、来源）
- `WordBook` - 词库（CET4/CET6/雅思等）
- `WordFamily` - 词族派生树（词根原型、成员列表）
- `LearningRecord` - 学习记录（SM-2 参数）
- `TypingSession` - 打字练习会话（WPM/正确率/连击/章节）
- `ListeningSession` - 听力练习会话（模式/正确数/时长）
- `Checkin` - 打卡记录
- `Achievement` - 成就定义
- `LearningPath` → `Unit` → `Level` → `Exercise` - 闯关体系

## 六、部署方案

### 开发环境
```bash
docker compose up -d    # 启动 PostgreSQL + API + Web
```

### 生产部署（推荐）
| 组件 | 平台 | 说明 |
|------|------|------|
| 前端 | Cloudflare Pages / Vercel | Next.js 静态 + SSR |
| 后端 | Railway / Render / VPS | Express API |
| 数据库 | Neon / Supabase | 免费 PostgreSQL |
| 域名 | Cloudflare DNS | `zqbjsy.xyz` |

### 词库导入
```bash
npx tsx scripts/importQwertyDicts.ts    # 导入全部 6 个词库
```
