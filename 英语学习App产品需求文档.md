# LightWords / 简词 - 英语学习App产品需求文档

## 一、产品概述

本产品是一款融合"不背单词"与"多邻国"两大经典英语学习应用核心优势的个人英语学习工具。通过科学的记忆算法与趣味化的学习机制，帮助用户在轻松愉快的氛围中掌握英语词汇、提升语言综合能力。

### 1.1 产品定位

- **目标用户**：想要系统学习英语的成年人，包括在校学生、职场人士、留学备考群体
- **核心价值**：让英语学习变得科学、高效、有趣
- **差异化特点**：结合词汇深度记忆与游戏化闯关模式

### 1.2 核心特色

| 特色模块 | 不背单词特色 | 多邻国特色 | 新增特色 |
|---------|-------------|-----------|---------|
| 词汇学习 | 词根词缀记忆法、真题语境 | 基础词汇闯关 | 图背单词、用户生成助记 |
| 复习机制 | 艾宾浩斯遗忘曲线 | 单元回顾 | 智能遗忘曲线、词文串学 |
| 学习方式 | 自主学习 | 游戏化闯关 | 社交学习（同桌、组队） |
| 激励系统 | 打卡统计 | 金币、成就、排行榜 | 刷题导出、试卷模式 |

## 三、核心功能详细说明

### 3.1 单词学习模块（参考不背单词）

#### 3.1.1 词根词缀记忆法

- **功能描述**：每个单词展示词根、词缀解析，帮助用户理解单词构成规律
- **展示内容**：
  - 词根含义及来源（拉丁语、希腊语等）
  - 词缀分类（前缀改变含义、后缀改变词性）
  - 同根词联想展示
  - 词源故事或记忆技巧

**示例展示**：

```
单词：transport
├── 词根：trans-（穿过）+ port（携带）
├── 记忆技巧："转移-携带" → 运输
├── 同根词：
│   ├── transport（运输）
│   ├── export（出口）
│   ├── import（进口）
│   └── portable（便携的）
└── 真题例句：The goods were transported by rail.
```

#### 3.1.2 语境记忆

- **功能描述**：每个单词配套多个真实语境例句
- **例句来源**：
  - 电影台词（标注电影名称）
  - 美剧片段
  - 新闻报道
  - 经典文学
  - 历年真题（标注年份、题型）
- **交互功能**：
  - 点击单词可查释义
  - 整句朗读（AI发音）
  - 例句收藏到生词本

#### 3.1.3 词组搭配

- **功能描述**：展示单词常用搭配和短语
- **内容类型**：
  - 动词短语（take off, break down）
  - 介词搭配（depend on, consist of）
  - 固定表达（by the way, in addition）
- **练习方式**：选词填空、连线匹配

#### 3.1.4 单词详情页

```
┌────────────────────────────────────┐
│        aboard                      │
│         [美][英][播放]              │
│         /ə'bɔːrd/                  │
├────────────────────────────────────┤
│  [词性] adv. 在船（飞机/火车）上    │
├────────────────────────────────────┤
│  【记忆技巧】                      │
│  a-（在...上）+ board（木板）       │
│  → 在船上                          │
├────────────────────────────────────┤
│  【真题例句】（2021高考）          │
│  They were the last to get aboard. │
│  他们是最后上船的。                │
├────────────────────────────────────┤
│  【词组搭配】                      │
│  go aboard 上船/飞机              │
│  climb aboard 攀上                │
├────────────────────────────────────┤
│  [认识] [模糊] [不认识] [添加至生词本]│
└────────────────────────────────────┘
```

### 3.2 图背单词（参考百词斩）

- **功能描述**：通过生动图片帮助用户快速建立单词与含义的强关联
- **图片来源**：
  - AI生成的语义图片
  - 真实场景图片库
  - 用户上传图片
- **图片类型**：
  - 实物图片（直接对应单词含义）
  - 联想图片（情景记忆）
  - 漫画/表情包（趣味记忆）
- **学习模式**：
  - 先看图片，再回忆单词
  - 图片+单词对照学习
  - 测试模式：看图片选单词
- **注意事项**：避免图片依赖，定期切换无图片模式

### 3.3 用户助记社区（参考墨墨背单词）

- **功能描述**：用户可以创建、分享、使用其他用户的记忆技巧
- **助记类型**：
  - 词缀分析（如：un-表示否定）
  - 谐音记忆（如：ambulance = 俺不能死）
  - 联想故事（用单词编小故事）
  - 思维导图（单词网络关系）
  - 词根词源讲解
- **社区功能**：
  - 点赞、收藏优质助记
  - 评论交流记忆心得
  - 举报低质量内容
  - 创建者获得金币奖励
- **助记排序**：
  - 按使用热度排序
  - 按评分排序
  - 个性化推荐

### 3.4 词文串学（参考扇贝单词）

- **功能描述**：将当日所学单词智能生成外刊同源短文，在语境中强化记忆
- **AI生成内容**：
  - 短文包含当日所学≥70%的单词
  - 难度与用户水平匹配
  - 主题多样化（新闻、科技、文化、生活）
- **学习功能**：
  - 点击高亮单词查看释义
  - 逐句朗读
  - 生词再次加入学习队列
  - 阅读理解练习

### 3.5 刷题与考试模式

- **功能描述**：从"会背"到"会考"的转化
- **试卷字体模式**：
  - 切换到考试字体（宋体、Times New Roman）
  - 界面简洁，无干扰元素
  - 适合刷题和模拟考试
- **导出功能**：
  - 导出PDF/Word文档
  - 按词库、章节、难度筛选
  - 含答案和解析
- **模拟考试**：
  - 限时答题
  - 实时计分
  - 成绩分析报告

### 3.6 社交学习系统

#### 3.6.1 同桌系统

- **功能描述**：找一个学习伙伴，互相督促
- **匹配方式**：
  - 随机匹配（同水平、同目标）
  - 邀请好友
  - 搜索用户
- **互动功能**：
  - 每日学习进度可见
  - 互相发送鼓励消息
  - 完成共同任务获得奖励
  - 解除/更换同桌

#### 3.6.2 组队打卡

- **功能描述**：3-5人组成小组，共同完成打卡目标
- **组队规则**：
  - 队长创建队伍
  - 设置打卡目标
  - 队员加入/退出
- **激励机制**：
  - 全组完成获得团队奖励
  - 小组排名
  - 连续组队天数统计
  - 掉落专属成就

### 3.7 闯关学习模式（参考多邻国）

#### 3.7.1 学习地图

- **功能描述**：可视化展示学习路径，类似游戏关卡地图
- **地图元素**：
  - 节点代表单元/课程
  - 连接线表示学习顺序
  - 锁头图标表示未解锁内容
  - 星星数量表示掌握程度（1-3星）
  - 当前进度指示器
- **地图分类**：
  - 基础词汇路径
  - 语法专项路径
  - 场景对话路径
  - 考试备考路径

#### 3.7.2 闯关机制

- **关卡结构**：每个关卡包含5-10个练习
- **练习类型**：
  - 选择正确翻译
  - 听音选词
  - 单词拼写
  - 句子排序
  - 看图说词
  - 填空补全
- **关卡评价**：
  - 完美通关（100%正确率）→ 3星 + 金币奖励
  - 优秀通关（≥80%）→ 2星 + 银币奖励
  - 勉强通关（≥60%）→ 1星 + 铜币奖励
  - 未通过（<60%）→ 消耗生命值，可重试

#### 3.7.3 生命值/体力系统

- **初始设置**：5颗心（可消耗道具恢复）
- **失败惩罚**：答错题扣除1颗心
- **恢复机制**：
  - 等待恢复（1颗心/30分钟）
  - 使用道具（ lingot / 金币购买）
  - 完成冒险获得
- **游戏结束**：心归零时需等待恢复或购买才能继续闯关

#### 3.7.4 连击系统

- **连击规则**：连续答对题目累积连击数
- **连击奖励**：
  - 5连击：额外10金币
  - 10连击：额外25金币
  - 20连击：额外50金币 + 特殊成就
- **连击中断**：答错题目连击归零

### 3.8 复习系统模块

#### 3.8.1 智能遗忘曲线（参考墨墨背单词）

- **算法实现**：基于用户历史记忆数据，绘制专属遗忘曲线
- **数据积累**：记录用户对每个单词的：
  - 认识/模糊/不认识操作
  - 答题反应时间
  - 错误次数
  - 掌握程度
- **复习节点**：
  - 动态计算，而非固定天数
  - 基于超过1200亿条用户记忆数据训练（参考墨墨背单词）
  - 精准预测每个单词的遗忘临界点
- **智能调整**：
  - 难记单词更频繁复习
  - 简单单词减少复习次数
  - 长期未忘单词延长复习间隔

#### 3.8.2 记忆曲线可视化

- **功能描述**：展示用户个人记忆数据
- **可视化内容**：
  - 个人专属遗忘曲线图表
  - 单词掌握度分布热力图
  - 学习效果对比（与平均水平）
- **数据洞察**：
  - 哪些单词最容易遗忘
  - 最佳学习时段分析
  - 学习效率评估

#### 3.8.3 复习模式

- **复习类型**：
  - 快速复习（今日待复习）
  - 深度复习（薄弱词汇强化）
  - 考前突击（高频考点词）
- **复习形式**：
  - 闪卡模式（先回忆后查看）
  - 测试模式（选择题/拼写）
  - 听写模式（音频听写）

#### 3.8.4 记忆分析

- **数据指标**：
  - 记忆强度分布图（强/中/弱）
  - 遗忘曲线图
  - 复习效率趋势
  - 遗忘风险预警
- **可视化展示**：折线图、饼图、热力图

### 3.9 听说读写训练

#### 3.9.1 听力训练

- **功能模块**：

  - 听音选句

#### 3.9.3 阅读理解

#### 3.9.4 写作练习

## 四、用户旅程设计

```
下载App → 注册账号 → 选择学习目标 
→ 测试当前水平 → 生成学习计划 
→ 完成新手教程 → 开始第一课
```

### 4.2 每日学习流程

```
打开App → 查看今日目标 
→ 完成单词学习（闯关模式） 
→ 进行听说读写练习 
→ 复习待复习单词 
→ 查看学习报告 → 打卡
```

### 4.3 学习循环

```
学习新词 → 即时测试 → 间隔复习 
→ 深度复习 → 掌握检测 → 融入技能训练
```

## 五、技术架构方案

### 5.1 技术选型

#### 推荐方案：React Native + React

| 平台 | 技术栈 | 优势 |
|-----|--------|------|
| iOS/Android | **React Native** | 原生性能，跨平台，原生API丰富 |
| Web | **React** | 组件复用率高，生态成熟 |

**优点**：
- ✅ 一套代码库（Web和App共享业务逻辑）

## 八、总结

本产品通过融合不背单词的深度词汇学习理念与多邻国的趣味闯关机制，同时结合了百词斩的图背单词、墨墨背单词的智能遗忘曲线、扇贝单词的词文串学等优秀功能，打造一款集众家之长的英语学习应用。重点突出以下核心价值：

1. **科学的记忆体系**：词根词缀+智能遗忘曲线，让记忆更持久
2. **趣味的学习体验**：闯关升级+图背单词+成就系统，让坚持更容易
3. **全面的能力训练**：听说读写+AI对话+词文串学，让应用更实用
4. **强大的社区生态**：用户助记+同桌+组队打卡，让学习不孤单
5. **清晰的数据反馈**：可视化报告+进度追踪，让学习有方向
